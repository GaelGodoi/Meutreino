# Meu Treino — MVP

Aplicativo PWA de treino personalizado focado em:
- simplicidade
- adaptação ao usuário
- treino  humanizado
- rotina  real

---

# Objetivo do projeto

O Meu Treino nasceu com a ideia de fugir dos apps fitness genéricos e complexos.

A proposta é:
- adaptar o treino à rotina real da pessoa
- gerar treinos coerentes
- manter interface simples
- facilitar consistência
- tornar o treino mais humano

---

# Principais diferenciais

## Treino adaptado ao usuário

O app considera:
- objetivo
- nível
- foco corporal
- disponibilidade
- tempo de treino
- limitações

# Estrutura inteligente de treino

Atualmente suporta:
- ABC
- ABCD
- ABCDE

Com lógica adaptativa entre:
- grupos musculares
- volume
- frequência
- recuperação

---

# Base inteligente de exercícios

Os exercícios possuem:
- nome consolidado
- variações específicas
- pegadas
- grupos musculares
- detalhes
- busca automática no YouTube

Exemplo:
- Puxada alta pronada aberta
- Crossover polia baixa
- Rosca direta barra W
